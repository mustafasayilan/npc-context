# Trademark Policy

This policy explains how the VamaoLabs name and the VamaoLabs NPC Context product name may be used.

## Allowed Uses

You may use the names "VamaoLabs" and "VamaoLabs NPC Context" to accurately describe the origin of the original project, for example:

- "This project is based on VamaoLabs NPC Context."
- "Compatible with VamaoLabs NPC Context."
- "Forked from VamaoLabs NPC Context."

## Restricted Uses

You may not, without written permission:

- publish a modified version as an official VamaoLabs release
- use VamaoLabs names or logos in a way that implies endorsement
- use a confusingly similar project name for a redistributed fork
- remove required copyright, license, trademark, or attribution notices

## Forks

Forks are welcome under the Apache-2.0 license. Modified forks should use a clearly different project name and should keep attribution to the original project.

This policy is not legal advice.

# Support

Thanks for trying VamaoLabs NPC Context.

## Where to Ask

- Use [GitHub Discussions](https://github.com/mustafasayilan/npc-context/discussions) for setup questions, ideas, and general feedback.
- Use [GitHub Issues](https://github.com/mustafasayilan/npc-context/issues) for reproducible bugs and focused feature requests.
- Use the repository security flow for private security reports. See [SECURITY.md](SECURITY.md).

## Before Opening an Issue

Please include:

- operating system and shell
- Node.js version
- install method
- command you ran
- short output or error message
- whether this is project-level or global setup

Please do not paste secrets, private source code, tokens, or credentials into public issues.

## Maintenance Scope

This project is maintained as open source software. I will do my best to respond, but support is best-effort and not a paid support contract.
